Create a token

For security, tokens are minted over SSH — never from the dashboard. On your server:

helmsman token mint \
  --scopes status:read,metrics:read,deploy:write:my-app \
  --cidrs 203.0.113.0/24 \
  --ttl 720h

The token value is shown once. It appears here so you can review and revoke it.
